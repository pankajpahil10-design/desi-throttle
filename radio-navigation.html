1. VOR stands for:

a. Visual Omni Range
b. VHF Omni Range
c. VHF Omnidirectional Range
d. Visual Omnidirectional Radio
Answer: c. VHF Omnidirectional Range

VOR operates 108-118 MHz, provides radial TO/FROM station.

2. DME measures:

a. Magnetic heading
b. Slant range distance
c. Ground speed
d. Altitude
Answer: b. Slant range distance

DME measures line-of-sight distance to station.

3. ILS provides:

a. Distance only
b. Direction only
c. Both direction and distance
d. Weather information
Answer: c. Both direction and distance

Localizer = horizontal, Glideslope = vertical, Markers = distance.

4. NDB operates in:

a. VHF band
b. UHF band
c. LF/MF band
d. SHF band
Answer: c. LF/MF band (190-1750 kHz)

NDB used for ADF navigation.

5. GPS uses:

a. Ground-based stations
b. Satellite constellation
c. Radio waves only
d. Radar
Answer: b. Satellite constellation (24+ satellites)

GPS provides PVT - Position, Velocity, Time.

6. ILS localizer frequency range:

a. 108-112 MHz
b. 108-118 MHz
c. 108-111.95 MHz
d. 112-118 MHz
Answer: c. 108-111.95 MHz

ILS Localizer = 108-111.95 MHz (odd decimals).

7. Glideslope operates at:

a. 328-336 MHz
b. 329-335 MHz
c. 325-330 MHz
d. 330-340 MHz
Answer: b. 329-335 MHz

Glideslope in UHF band.

8. Marker beacons transmit at:

a. 75 MHz
b. 108 MHz
c. 118 MHz
d. 1300 MHz
Answer: a. 75 MHz

OM at 400Hz, MM at 1300Hz, IM at 3000Hz.

9. VOR accuracy:

a. ¬±1¬∞
b. ¬±3¬∞
c. ¬±5¬∞
d. ¬±10¬∞
Answer: c. ¬±5¬∞

VOR accuracy is ¬±5¬∞ in service volume.

10. DME frequency paired with VOR:

a. VOR + 63 MHz
b. VOR + 1 MHz
c. VOR X channel
d. Automatically selected
Answer: c. VOR X channel (VOR freq + 63 MHz)

DME automatically tuned with VOR frequency.

11. Radar altimeter measures:

a. Pressure altitude
b. Height above terrain
c. Barometric altitude
d. GPS altitude
Answer: b. Height above terrain (RADAR ALT)

Radio altimeter measures true height above ground.

12. Secondary radar requires:

a. Ground transponder
b. Aircraft transponder
c. Both aircraft and ground
d. No transponder
Answer: b. Aircraft transponder

SSR/TCAS require aircraft transponder.

13. Mode A provides:

a. Altitude
b. Identity code
c. Call sign
d. Speed
Answer: b. Identity code (4-digit squawk)

Mode A = identity code only.

14. Mode C provides:

a. Identity only
b. Identity + altitude
c. Speed
d. Heading
Answer: b. Identity + altitude (pressure altitude)

Mode C = identity and altitude reporting.

15. TCAS I provides:

a. Traffic advisory only
b. Resolution advisory
c. Traffic + TA
d. No warning
Answer: c. Traffic advisory only

TCAS I = Traffic Advisory only. TCAS II = TA + RA.

16. RNAV is:

a. Required Navigation
b. Area Navigation
c. Radio Navigation
d. Relative Navigation
Answer: b. Area Navigation

RNAV allows arbitrary flight paths between waypoints.

17. RNP is:

a. Required Navigation Performance
b. Radio Navigation Position
c. Reduced Navigation Capability
d. Runway Navigation Position
Answer: a. Required Navigation Performance

RNP specifies navigation accuracy in NM (e.g., RNP 0.5 = ¬±0.5 NM).

18. LORAN operates in:

a. VHF
b. UHF
c. LF/HF
d. SHF
Answer: c. LF/HF (90-110 kHz)

LORAN-C operates in Low Frequency band.

19. Inertial Navigation uses:

a. GPS signals
b. Ground stations
c. Accelerometers and gyros
d. Radio waves
Answer: c. Accelerometers and gyros

INS/IRS uses inertial reference units (accelerometers + gyros).

20. Waypoint is:

a. Airport
b. Navigation fix defined by coordinates
c. VOR station
d. NDB station
Answer: b. A defined geographical point used for navigation

Waypoints can be VOR, NDB, or coordinate-based.

21. VORTAC provides:

a. VOR only
b. DME only
c. VOR + DME (co-located)
d. ILS
Answer: c. VOR + DME co-located

VORTAC = VOR + TACAN (military DME).

22. ADF indicates:

a. Distance to station
b. Bearing TO NDB
c. Ground speed
d. Altitude
Answer: b. Bearing TO NDB (relative bearing)

ADF shows relative bearing to NDB station.

23. To convert relative bearing to magnetic bearing:

a. Add heading
b. Add relative to heading
c. Subtract heading
d. Multiply by 2
Answer: b. RB + MH = MB

Magnetic bearing = Relative bearing + Magnetic heading.

24. RVR stands for:

a. Runway Visual Range
b. Runway Visibility Range
c. Required Visual Range
d. Radio Visual Range
Answer: a. Runway Visual Range

RVR = visibility along runway measured by transmissometer.

25. Minimum RVR for CAT I landing:

a. 200m
b. 350m
c. 550m
d. 1200m
Answer: c. 550m (with DH 200ft)

CAT I = RVR 550m + DH 200ft + lighting.

26. MLS stands for:

a. Microwave Landing System
b. Medium Landing System
c. Military Landing System
d. Main Landing System
Answer: a. Microwave Landing System

MLS provides precision approach in C-band.

27. Frequency for VOR:

a. 108-118 MHz
b. 108-112 MHz
c. 112-118 MHz
d. 108-136 MHz
Answer: a. 108-118 MHz

VOR occupies 108-118 MHz (first 0.5 MHz for ILS).

28. Doppler VOR uses:

a. Ground station rotation
b. Frequency shift principle
c. Phase comparison
d. Time delay
Answer: b. Doppler shift principle

Doppler VOR uses rotating antenna to create Doppler shift.

29. ILS category determines:

a. Frequency
b. Decision Height and RVR minimums
c. Runway length
d. Transmitter power
Answer: b. Decision Height and RVR minimums

CAT I/II/III determined by DH and RVR minimums.

30. MLS provides:

a. Azimuth only
b. Elevation only
c. Azimuth + Elevation + DME
d. Direction only
Answer: c. Azimuth + Elevation + DME

MLS provides full 3D approach guidance.

31. Basic en-route navigation uses:

a. Dead reckoning
b. Pilotage
c. Radio navigation
d. All of above
Answer: d. All - DR, Pilotage, Radio Nav

En-route uses combination of methods.

32. WPT on FMS means:

a. Waypoint
b. Wind
c. Weight
d. Weather
Answer: a. Waypoint

WPT = Waypoint identifier on FMS.

33. ETP is:

a. Estimated Time enroute
b. Equal Time Point
c. Estimated Time Position
d. Emergency Transfer Point
Answer: b. Equal Time Point

ETP = point where time to destination = time to alternate.

34. PNR is:

a. Point of No Return
b. Position Navigation Report
c. Primary Navigation Reference
d. Pilot Navigation Route
Answer: a. Point of No Return

PNR = point where remaining fuel = fuel to destination.

35. TACAN provides:

a. VOR only
b. DME only
c. Bearing + DME
d. ILS
Answer: c. Bearing + DME (military equivalent of VOR+DME)

TACAN = bearing (like VOR) + DME.

36. Maximum DME range:

a. 50 nm
b. 100 nm
c. 200 nm
d. 500 nm
Answer: c. ~200 nm (line of sight)

DME range is line-of-sight, typically 200+ nm at altitude.

37. ATC transponder code 7700:

a. VFR
b. Emergency
c. Radio failure
d. Hijack
Answer: b. Emergency (distress)

7700 = Emergency. 7500 = Hijack. 7600 = Radio failure.

38. ATC transponder code 7500:

a. VFR
b. Emergency
c. Radio failure
d. Hijack
Answer: d. Hijack

7500 = Unlawful Interference (Hijack).

39. ATC transponder code 7600:

a. VFR
b. Emergency
c. Radio failure
d. Hijack
Answer: c. Radio failure (NORDO)

7600 = Loss of communications.

40. ATC code 2000:

a. Special VFR
b. No transponder
c. General VFR
d. Military
Answer: b. Non-transponder aircraft

2000 = Code for aircraft without transponder.
